# Enhancer
Super resolution convolutional neural network(SRCNN) based on Tensorflow framework.

## Prerequisites
 * Tensorflow >=1.3.0
 * Scipy version > 0.18 ('mode' option from scipy.misc.imread function)

## Instruction
For training, `python main.py`

## References
